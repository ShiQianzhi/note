Python 是一种**高级、通用、解释型**的编程语言，由 Guido van Rossum 于 1991 年创建。以下是 Python 的核心特点和应用领域：

## 主要特点

1. **简洁优雅**：语法简洁，代码可读性高，强调"可读性至关重要"
2. **跨平台**：支持 Windows、macOS、Linux 等多种操作系统
3. **动态类型**：无需声明变量类型，运行时自动推断
4. **面向对象**：支持面向对象编程，同时也支持函数式、过程式等多种编程范式
5. **丰富的标准库**：内置了大量模块，覆盖文件处理、网络通信、正则表达式等
6. **强大的第三方库**：NumPy、Pandas、TensorFlow、Django、Flask 等

## 应用领域

- **Web 开发**：Django、Flask 等框架
- **数据科学与机器学习**：TensorFlow、PyTorch、Scikit-learn
- **自动化运维**：脚本编写、系统管理
- **科学计算**：NumPy、SciPy
- **爬虫开发**：Requests、Scrapy
- **桌面应用**：PyQt、Tkinter
- **游戏开发**：Pygame

## 设计哲学

Python 的设计遵循"Python 之禅"（The Zen of Python），强调：
- 优美胜于丑陋
- 明了胜于晦涩
- 简洁胜于复杂
- 可读性很重要

## 示例代码

```python
print("Hello, World!")

def greet(name):
    return f"Hello, {name}!"

print(greet("Python"))
```
