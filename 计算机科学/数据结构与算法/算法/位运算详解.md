# 位运算详解

## 前言

位运算（Bitwise Operation）是直接对二进制位进行操作的运算方式。在计算机底层，所有数据都以二进制形式存储，位运算能够直接操作这些二进制位，具有极高的执行效率。位运算在算法设计、系统编程、硬件控制、加密解密等领域有着广泛的应用。

本文将从基础概念出发，深入讲解各种位运算符的原理和应用，结合大量代码示例，帮助读者全面掌握位运算技术。

---

## 1. 二进制基础

### 1.1 二进制表示

计算机中，所有数据都以二进制（Binary）形式存储，即由 0 和 1 组成。每一位称为一个 **bit**（比特），8 位称为一个 **byte**（字节）。

```
十进制:     15
二进制:    1111
十六进制:   0xF
```

### 1.2 有符号数与无符号数

**无符号数**：所有位都用于表示数值大小，取值范围从 0 到 2^n - 1。

**有符号数**：最高位作为符号位（0 表示正数，1 表示负数），其余位表示数值大小。负数采用 **补码** 表示。

```c
// 以 4 位二进制为例
无符号数: 0000 ~ 1111  → 0 ~ 15

有符号数: 
  正数: 0000 ~ 0111  → 0 ~ 7
  负数: 1000 ~ 1111  → -8 ~ -1
```

### 1.3 原码、反码、补码

**原码**：最高位为符号位，其余位为数值位。
```
+5: 0000 0101
-5: 1000 0101
```

**反码**：正数的反码等于原码；负数的反码是原码除符号位外取反。
```
+5: 0000 0101
-5: 1111 1010
```

**补码**：正数的补码等于原码；负数的补码是反码加 1。
```
+5: 0000 0101
-5: 1111 1011
```

> **为什么使用补码？**
> - 统一正负数的加减法运算
> - 0 的表示唯一
> - 简化硬件实现

## 2. 位运算符速查表

| 运算符 | 名称 | 符号 | 运算规则 | 示例 |
|--------|------|------|----------|------|
| `&` | 按位与 | AND | 全 1 为 1，否则为 0 | `1010 & 1100 = 1000` |
| `\|` | 按位或 | OR | 全 0 为 0，否则为 1 | `1010 \| 1100 = 1110` |
| `^` | 按位异或 | XOR | 相同为 0，不同为 1 | `1010 ^ 1100 = 0110` |
| `~` | 按位取反 | NOT | 0 变 1，1 变 0 | `~1010 = 0101` |
| `<<` | 左移 | SHL | 各位左移，低位补 0 | `1010 << 1 = 0100` |
| `>>` | 右移 | SHR | 各位右移，高位补符号位 | `1010 >> 1 = 0101` |

## 3. 按位与运算 (`&`)

### 3.1 运算规则

只有当两个对应位都为 1 时，结果才为 1；否则为 0。

```
  1010  (10)
& 1100  (12)
------
  1000  (8)
```

### 3.2 核心应用

**应用 1：检查某一位是否为 1**

```c
#include <stdio.h>

int main() {
    int num = 0b1010;  // 10
    int bit_pos = 3;   // 检查第 3 位（从 0 开始计数）
    
    if (num & (1 << bit_pos)) {
        printf("第 %d 位是 1\n", bit_pos);
    } else {
        printf("第 %d 位是 0\n", bit_pos);
    }
    
    return 0;
}
```

**应用 2：将某一位设置为 0**

```c
int clear_bit(int num, int bit_pos) {
    return num & ~(1 << bit_pos);
}
```

**应用 3：提取某些位**

```c
// 提取低 8 位
int low_byte = num & 0xFF;

// 提取高 8 位
int high_byte = (num >> 8) & 0xFF;
```

**应用 4：判断奇偶性**

```c
int is_even(int num) {
    return (num & 1) == 0;
}

int is_odd(int num) {
    return (num & 1) == 1;
}
```

**应用 5：向下取整到 2 的幂次**

```c
// 向下取整到最近的 8 的倍数
int round_down_8(int num) {
    return num & ~0x7;  // 等价于 num & (-8)
}
```

## 4. 按位或运算 (`|`)

### 4.1 运算规则

只要有一个对应位为 1，结果就为 1；否则为 0。

```
  1010  (10)
| 1100  (12)
------
  1110  (14)
```

### 4.2 核心应用

**应用 1：将某一位设置为 1**

```c
int set_bit(int num, int bit_pos) {
    return num | (1 << bit_pos);
}
```

**应用 2：合并标志位**

```c
#define FLAG_READ  0x01
#define FLAG_WRITE 0x02
#define FLAG_EXEC  0x04

// 设置多个标志
int flags = FLAG_READ | FLAG_WRITE;  // 0x03
```

**应用 3：强制转换为正数**

```c
// 对于负数，返回其绝对值（仅适用于 int）
int to_positive(int num) {
    return num & 0x7FFFFFFF;
}
```

**应用 4：RGB 颜色值合并**

```c
int red = 0xFF;
int green = 0x80;
int blue = 0x00;

int color = (red << 16) | (green << 8) | blue;
// color = 0xFF8000
```

## 5. 按位异或运算 (`^`)

### 5.1 运算规则

两个对应位相同时为 0，不同时为 1。

```
  1010  (10)
^ 1100  (12)
------
  0110  (6)
```

### 5.2 异或运算的数学性质

| 性质 | 说明 | 示例 |
|------|------|------|
| 交换律 | `a ^ b = b ^ a` | `3 ^ 5 = 5 ^ 3` |
| 结合律 | `(a ^ b) ^ c = a ^ (b ^ c)` | `(1 ^ 2) ^ 3 = 1 ^ (2 ^ 3)` |
| 自反性 | `a ^ a = 0` | `5 ^ 5 = 0` |
| 恒等律 | `a ^ 0 = a` | `5 ^ 0 = 5` |
| 与或的关系 | `a ^ b = (a \| b) & ~(a & b)` | - |

### 5.3 核心应用

**应用 1：交换两个变量（无需临时变量）**

```c
void swap(int *a, int *b) {
    *a = *a ^ *b;
    *b = *a ^ *b;
    *a = *a ^ *b;
}
```

**原理分析：**
```
初始: a = x, b = y
步骤1: a = x ^ y, b = y
步骤2: b = (x ^ y) ^ y = x ^ (y ^ y) = x ^ 0 = x
步骤3: a = (x ^ y) ^ x = (x ^ x) ^ y = 0 ^ y = y
最终: a = y, b = x
```

**应用 2：找出唯一出现一次的数**

```c
// 数组中只有一个数出现一次，其他数都出现两次
int single_number(int nums[], int n) {
    int result = 0;
    for (int i = 0; i < n; i++) {
        result ^= nums[i];
    }
    return result;
}
```

**原理分析：**
- 相同的数异或结果为 0
- 0 异或任何数等于该数本身
- 所有数异或后，成对的数相互抵消，剩下的就是唯一出现一次的数

**应用 3：找出两个唯一出现一次的数**

```c
void find_two_unique(int nums[], int n, int *a, int *b) {
    int xor_result = 0;
    for (int i = 0; i < n; i++) {
        xor_result ^= nums[i];
    }
    
    // 找到最右边的 1
    int rightmost_bit = xor_result & -xor_result;
    
    *a = 0;
    *b = 0;
    for (int i = 0; i < n; i++) {
        if (nums[i] & rightmost_bit) {
            *a ^= nums[i];
        } else {
            *b ^= nums[i];
        }
    }
}
```

**应用 4：快速判断两个数是否相等**

```c
int is_equal(int a, int b) {
    return (a ^ b) == 0;
}
```

**应用 5：翻转某些位**

```c
int flip_bits(int num, int mask) {
    return num ^ mask;
}
```

## 6. 按位取反运算 (`~`)

### 6.1 运算规则

将每一位取反：0 变为 1，1 变为 0。

```
~1010 = 0101
```

### 6.2 注意事项

在有符号整数中，取反后结果是负数（因为最高位变为 1）。

```c
#include <stdio.h>

int main() {
    unsigned char a = 0b1010;  // 10
    printf("~%d = %u\n", a, ~a);  // ~10 = 245
    
    char b = 0b1010;  // 10
    printf("~%d = %d\n", b, ~b);  // ~10 = -11（补码表示）
    
    return 0;
}
```

### 6.3 核心应用

**应用 1：创建掩码**

```c
// 创建低 n 位全为 1 的掩码
int mask_n_bits(int n) {
    return (1 << n) - 1;
}

// 创建高 n 位全为 1 的掩码
int mask_high_bits(int n) {
    return ~((1 << (32 - n)) - 1);
}
```

**应用 2：计算绝对值（针对负数）**

```c
int abs_value(int num) {
    int mask = num >> 31;
    return (num ^ mask) - mask;
}
```

**原理分析：**
- 对于正数：`mask = 0`，`(num ^ 0) - 0 = num`
- 对于负数：`mask = -1`，`(num ^ -1) - (-1) = ~num + 1 = -num`

## 7. 左移运算 (`<<`)

### 7.1 运算规则

将所有位向左移动指定的位数，低位补 0，高位溢出舍弃。

```
1010 << 1 = 0100  (10 << 1 = 20)
1010 << 2 = 0000  (10 << 2 = 40)
```

### 7.2 数学性质

`x << n` 等价于 `x * 2^n`（当不发生溢出时）。

### 7.3 核心应用

**应用 1：快速乘法**

```c
int multiply_by_power_of_two(int num, int power) {
    return num << power;
}

// 乘以 8
int result = num << 3;
```

**应用 2：创建位掩码**

```c
int create_mask(int bit_pos) {
    return 1 << bit_pos;
}

// 创建第 5 位为 1 的掩码
int mask = 1 << 5;  // 0b100000
```

**应用 3：组合数据**

```c
// 将多个小数值组合成一个大整数
int year = 2024;
int month = 12;
int day = 25;

int date = (year << 9) | (month << 5) | day;
```

**应用 4：RGB 颜色提取**

```c
int color = 0xFF8000;
int red = (color >> 16) & 0xFF;
int green = (color >> 8) & 0xFF;
int blue = color & 0xFF;
```

## 8. 右移运算 (`>>`)

### 8.1 运算规则

将所有位向右移动指定的位数。

**逻辑右移**：高位补 0（适用于无符号数）
```
1010 >> 1 = 0101
```

**算术右移**：高位补符号位（适用于有符号数）
```
正数: 0101 >> 1 = 0010
负数: 1010 >> 1 = 1101（补码表示，保持符号不变）
```

### 8.2 数学性质

`x >> n` 等价于 `floor(x / 2^n)`（对于有符号数）。

### 8.3 核心应用

**应用 1：快速除法**

```c
int divide_by_power_of_two(int num, int power) {
    return num >> power;
}

// 除以 8
int result = num >> 3;
```

**应用 2：获取符号位**

```c
int get_sign(int num) {
    return (num >> 31) & 1;  // 0 表示正数，1 表示负数
}
```

**应用 3：向下取整到 2 的幂次**

```c
int floor_to_power_of_two(int num) {
    return num >> 1 << 1;  // 向下取整到偶数
}
```

**应用 4：算术平均**

```c
// 计算 (a + b) / 2，避免溢出
int average(int a, int b) {
    return (a & b) + ((a ^ b) >> 1);
}
```

**原理分析：**
```
(a & b) 是 a 和 b 相同位的和（每个相同位贡献 2 倍的值）
(a ^ b) 是 a 和 b 不同位的和
(a ^ b) >> 1 等价于除以 2
两者相加就是 (a + b) / 2
```

## 9. 位运算技巧汇总

### 9.1 判断技巧

| 操作 | 代码 | 说明 |
|------|------|------|
| 判断奇偶 | `num & 1` | 结果为 1 是奇数，0 是偶数 |
| 判断是否为 2 的幂 | `(num & (num - 1)) == 0` | 只有一个 1 的数是 2 的幂 |
| 判断符号 | `num >> 31` | 正数为 0，负数为 -1 |
| 判断相等 | `(a ^ b) == 0` | 异或结果为 0 表示相等 |

### 9.2 计算技巧

| 操作 | 代码 | 说明 |
|------|------|------|
| 乘以 2 | `num << 1` | 左移 1 位 |
| 除以 2 | `num >> 1` | 右移 1 位 |
| 取绝对值 | `(num ^ (num >> 31)) - (num >> 31)` | 利用补码性质 |
| 两数交换 | `a ^= b; b ^= a; a ^= b` | 无需临时变量 |
| 算术平均 | `(a & b) + ((a ^ b) >> 1)` | 避免溢出 |

### 9.3 位操作技巧

| 操作 | 代码 | 说明 |
|------|------|------|
| 设置第 n 位为 1 | `num \| (1 << n)` | 或运算 |
| 设置第 n 位为 0 | `num & ~(1 << n)` | 与运算 + 取反 |
| 翻转第 n 位 | `num ^ (1 << n)` | 异或运算 |
| 检查第 n 位 | `num & (1 << n)` | 与运算 |
| 提取低 n 位 | `num & ((1 << n) - 1)` | 掩码与运算 |

## 10. 位运算在算法中的应用

### 10.1 位掩码枚举子集

```c
// 枚举集合的所有子集
void enumerate_subsets(int mask) {
    for (int subset = mask; subset >= 0; subset = (subset - 1) & mask) {
        printf("%d ", subset);
        if (subset == 0) break;
    }
}
```

**示例**：`mask = 0b101`（5），输出：`5 4 1 0`

### 10.2 状态压缩 DP

```c
// 旅行商问题（TSP）状态压缩
int tsp(int graph[][N], int n) {
    int dp[1 << N][N];
    memset(dp, INF, sizeof(dp));
    
    for (int i = 0; i < n; i++) {
        dp[1 << i][i] = 0;
    }
    
    for (int mask = 0; mask < (1 << n); mask++) {
        for (int u = 0; u < n; u++) {
            if (!(mask & (1 << u))) continue;
            for (int v = 0; v < n; v++) {
                if (!(mask & (1 << v))) {
                    dp[mask | (1 << v)][v] = min(dp[mask | (1 << v)][v], 
                                                  dp[mask][u] + graph[u][v]);
                }
            }
        }
    }
    
    return dp[(1 << n) - 1][0];
}
```

### 10.3 快速幂运算

```c
int fast_power(int base, int exponent) {
    int result = 1;
    while (exponent > 0) {
        if (exponent & 1) {
            result *= base;
        }
        base *= base;
        exponent >>= 1;
    }
    return result;
}
```

**原理分析**：
- 将指数转换为二进制
- 每一位代表是否需要乘上对应的幂次
- 时间复杂度：O(log n)

### 10.4 格雷码生成

```c
// 生成 n 位格雷码
void generate_gray_code(int n) {
    for (int i = 0; i < (1 << n); i++) {
        int gray = i ^ (i >> 1);
        printf("%d ", gray);
    }
}
```

**原理**：第 i 个格雷码 = i ^ (i >> 1)

### 10.5 最长连续 1 的个数

```c
int longest_ones(int num) {
    int max_count = 0;
    int count = 0;
    
    while (num > 0) {
        if (num & 1) {
            count++;
            max_count = max(max_count, count);
        } else {
            count = 0;
        }
        num >>= 1;
    }
    
    return max_count;
}
```

## 11. 位运算在系统编程中的应用

### 11.1 文件权限控制

```c
#include <stdio.h>

#define S_IRUSR 0400  // 用户读权限
#define S_IWUSR 0200  // 用户写权限
#define S_IXUSR 0100  // 用户执行权限
#define S_IRGRP 0040  // 组读权限

int main() {
    int mode = S_IRUSR | S_IWUSR | S_IRGRP | S_IROTH;
    // mode = 0644 (用户读写，组读，其他读)
    
    printf("权限模式: %o\n", mode);
    
    // 检查是否有写权限
    if (mode & S_IWUSR) {
        printf("用户有写权限\n");
    }
    
    // 添加执行权限
    mode |= S_IXUSR;
    printf("新权限模式: %o\n", mode);
    
    return 0;
}
```

### 11.2 硬件寄存器操作

```c
#include <stdint.h>

// 模拟硬件寄存器
#define GPIO_BASE 0x10000000

#define GPIO_DIR_OFFSET  0x00
#define GPIO_DATA_OFFSET 0x04

#define PIN_0 (1 << 0)
#define PIN_1 (1 << 1)
#define PIN_2 (1 << 2)

int main() {
    volatile uint32_t *gpio_dir = (uint32_t *)(GPIO_BASE + GPIO_DIR_OFFSET);
    volatile uint32_t *gpio_data = (uint32_t *)(GPIO_BASE + GPIO_DATA_OFFSET);
    
    // 设置 PIN_0 和 PIN_1 为输出模式
    *gpio_dir |= (PIN_0 | PIN_1);
    
    // 设置 PIN_0 为高电平
    *gpio_data |= PIN_0;
    
    // 设置 PIN_1 为低电平
    *gpio_data &= ~PIN_1;
    
    // 翻转 PIN_0 的状态
    *gpio_data ^= PIN_0;
    
    return 0;
}
```

## 12. 位运算在加密解密中的应用

### 12.1 简单的 XOR 加密

```c
#include <stdio.h>
#include <string.h>

void xor_cipher(char *data, int length, char key) {
    for (int i = 0; i < length; i++) {
        data[i] ^= key;
    }
}

int main() {
    char message[] = "Hello, World!";
    char key = 'X';
    
    printf("原始消息: %s\n", message);
    
    // 加密
    xor_cipher(message, strlen(message), key);
    printf("加密后: %s\n", message);
    
    // 解密（再次异或相同的密钥）
    xor_cipher(message, strlen(message), key);
    printf("解密后: %s\n", message);
    
    return 0;
}
```

### 12.2 RC4 算法中的位操作

```c
#include <stdio.h>
#include <string.h>

void rc4_init(unsigned char *s, unsigned char *key, int key_len) {
    int i, j = 0;
    
    for (i = 0; i < 256; i++) {
        s[i] = i;
    }
    
    for (i = 0; i < 256; i++) {
        j = (j + s[i] + key[i % key_len]) & 0xFF;
        unsigned char temp = s[i];
        s[i] = s[j];
        s[j] = temp;
    }
}

unsigned char rc4_next_byte(unsigned char *s, int *i, int *j) {
    *i = (*i + 1) & 0xFF;
    *j = (*j + s[*i]) & 0xFF;
    
    unsigned char temp = s[*i];
    s[*i] = s[*j];
    s[*j] = temp;
    
    return s[(s[*i] + s[*j]) & 0xFF];
}

void rc4_encrypt(unsigned char *data, int length, unsigned char *key, int key_len) {
    unsigned char s[256];
    int i = 0, j = 0;
    
    rc4_init(s, key, key_len);
    
    for (int k = 0; k < length; k++) {
        data[k] ^= rc4_next_byte(s, &i, &j);
    }
}

int main() {
    unsigned char message[] = "Hello, RC4!";
    unsigned char key[] = "secret_key";
    
    printf("原始消息: %s\n", message);
    
    rc4_encrypt(message, strlen((char *)message), key, strlen((char *)key));
    printf("加密后: ");
    for (int i = 0; i < strlen((char *)message); i++) {
        printf("%02X ", message[i]);
    }
    printf("\n");
    
    rc4_encrypt(message, strlen((char *)message), key, strlen((char *)key));
    printf("解密后: %s\n", message);
    
    return 0;
}
```

## 13. 位运算性能分析

### 13.1 位运算 vs 算术运算

| 操作 | 算术运算 | 位运算 | 性能差异 |
|------|----------|--------|----------|
| 乘以 2 | `a * 2` | `a << 1` | 位运算更快 |
| 除以 2 | `a / 2` | `a >> 1` | 位运算更快 |
| 判断奇偶 | `a % 2 == 0` | `(a & 1) == 0` | 位运算更快 |
| 判断相等 | `a == b` | `(a ^ b) == 0` | 相当 |

### 13.2 现代编译器优化

现代编译器（如 GCC、Clang）会自动将一些算术运算优化为位运算：

```c
int result = num * 8;
// 编译器可能优化为: result = num << 3;
```

但手动使用位运算可以：
1. 明确表达程序员的意图
2. 避免编译器优化失败的情况
3. 在嵌入式系统中保证确定性行为

## 14. 常见陷阱与注意事项

### 14.1 有符号数的右移

```c
int a = -4;
int b = a >> 1;
// b = -2（算术右移，保持符号）
```

### 14.2 溢出问题

```c
int a = INT_MAX;
int b = a << 1;  // 未定义行为！
```

### 14.3 优先级问题

```c
// 错误：位运算优先级低于比较运算
if (x & 1 == 0) { ... }

// 正确：需要括号
if ((x & 1) == 0) { ... }
```

### 14.4 整数提升

```c
char c = 0xFF;
int result = ~c;
// result = 0xFFFFFF00（char 提升为 int 后取反）
```

### 14.5 无符号数的取反

```c
unsigned int a = 0x000000FF;
unsigned int b = ~a;
// b = 0xFFFFFF00（无符号取反）

int c = 0x000000FF;
int d = ~c;
// d = 0xFFFFFF00（在内存中，但作为有符号数是 -256）
```

## 15. 实践练习

### 练习 1：实现快速幂

```c
int fast_power(int base, int exponent) {
    int result = 1;
    while (exponent > 0) {
        if (exponent & 1) {
            result *= base;
        }
        base *= base;
        exponent >>= 1;
    }
    return result;
}
```

### 练习 2：找出数组中唯一出现一次的数

```c
int single_number(int nums[], int n) {
    int result = 0;
    for (int i = 0; i < n; i++) {
        result ^= nums[i];
    }
    return result;
}
```

### 练习 3：判断一个数是否为 2 的幂

```c
int is_power_of_two(int num) {
    return num > 0 && (num & (num - 1)) == 0;
}
```

### 练习 4：计算二进制中 1 的个数

```c
int count_ones(int num) {
    int count = 0;
    while (num > 0) {
        num &= (num - 1);
        count++;
    }
    return count;
}
```

### 练习 5：交换两个整数（无需临时变量）

```c
void swap(int *a, int *b) {
    *a ^= *b;
    *b ^= *a;
    *a ^= *b;
}
```

### 练习 6：实现 RGB 颜色值的提取与组合

```c
typedef struct {
    int r, g, b;
} RGB;

int rgb_to_int(RGB color) {
    return (color.r << 16) | (color.g << 8) | color.b;
}

RGB int_to_rgb(int color) {
    RGB result;
    result.r = (color >> 16) & 0xFF;
    result.g = (color >> 8) & 0xFF;
    result.b = color & 0xFF;
    return result;
}
```

**核心要点：**

1. **运算效率**：位运算直接操作二进制位，比算术运算更快
2. **内存紧凑**：位掩码可以用少量位表示多个状态
3. **代码简洁**：很多复杂操作可以用简洁的位运算实现
4. **应用广泛**：算法、系统编程、嵌入式、加密等领域都离不开位运算
