# 深度学习基础

## 什么是深度学习？

**深度学习（Deep Learning，DL）** 是机器学习的一个分支，它使用多层神经网络来模拟人脑的学习过程。

### "深度"的含义

"深度"指的是神经网络的层数。传统机器学习算法通常只有一层或几层，而深度学习模型可以有几十层甚至上百层。

### 深度学习的崛起原因

1. **数据量爆炸**：互联网的发展带来了海量数据
2. **计算能力提升**：GPU的普及大幅提升了计算速度
3. **算法创新**：ReLU激活函数、批归一化等技术的出现

## 神经网络基础

### 生物神经元启发

深度学习的灵感来源于人脑的神经元结构：

- **树突**：接收信号
- **细胞体**：处理信号
- **轴突**：输出信号

### 人工神经元

人工神经元是神经网络的基本单元：

```
输入 → 加权求和 → 加偏置 → 激活函数 → 输出
```

数学表达式：

```
z = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
a = σ(z)
```

其中：
- `x`：输入
- `w`：权重（Weight），表示输入的重要程度
- `b`：偏置（Bias），调节激活的难易程度
- `σ`：激活函数（Activation Function），决定是否激活

### 神经网络结构

一个完整的神经网络通常包含：

1. **输入层（Input Layer）**：接收原始数据
2. **隐藏层（Hidden Layer）**：对数据进行特征提取和变换
3. **输出层（Output Layer）**：产生最终输出

```
输入层 → 隐藏层1 → 隐藏层2 → ... → 隐藏层n → 输出层
```

## 激活函数

### 为什么需要激活函数？

如果没有激活函数，神经网络只是多层线性变换的组合，无法学习非线性关系。激活函数为网络引入了非线性，使其能够学习复杂的模式。

### 常用激活函数

#### 1. Sigmoid函数

```
σ(z) = 1 / (1 + e^(-z))
```

**特点**：
- 输出范围：(0, 1)
- 常用于二分类任务的输出层
- **问题**：梯度消失（当z绝对值很大时，梯度接近0）

#### 2. Tanh函数

```
tanh(z) = (e^z - e^(-z)) / (e^z + e^(-z))
```

**特点**：
- 输出范围：(-1, 1)
- 比Sigmoid收敛更快（因为输出中心化）
- **问题**：同样存在梯度消失问题

#### 3. ReLU函数

```
ReLU(z) = max(0, z)
```

**特点**：
- 当z > 0时，输出等于输入；当z ≤ 0时，输出为0
- **优点**：
  - 计算简单，速度快
  - 缓解了梯度消失问题
  - 稀疏激活，节省计算资源
- **缺点**：
  - 神经元可能"死亡"（当z始终≤0时，梯度为0，无法更新）

#### 4. Leaky ReLU

```
LeakyReLU(z) = max(αz, z)，其中α很小（如0.01）
```

**特点**：解决了ReLU的神经元死亡问题，当z ≤ 0时，保留一个小的梯度。

#### 5. Softmax函数

```
Softmax(zᵢ) = e^zᵢ / Σe^zⱼ
```

**特点**：
- 将输出转换为概率分布，所有输出之和为1
- 常用于多分类任务的输出层

## 前向传播

### 什么是前向传播？

前向传播是数据从输入层流向输出层的过程，计算每一层的输出。

### 矩阵表示

使用矩阵运算可以高效计算前向传播：

```
Z⁽¹⁾ = W⁽¹⁾X + b⁽¹⁾
A⁽¹⁾ = σ(Z⁽¹⁾)
Z⁽²⁾ = W⁽²⁾A⁽¹⁾ + b⁽²⁾
A⁽²⁾ = σ(Z⁽²⁾)
...
Z⁽L⁾ = W⁽L⁾A⁽L⁻¹⁾ + b⁽L⁾
A⁽L⁾ = σ(Z⁽L⁾)
```

其中：
- `Z`：加权求和后的结果
- `A`：经过激活函数后的输出
- `W`：权重矩阵
- `b`：偏置向量
- `L`：神经网络的层数

## 反向传播

### 什么是反向传播？

反向传播是计算损失函数对模型参数（权重和偏置）的梯度的过程，是训练神经网络的核心算法。

### 核心思想

反向传播基于**链式法则**，从输出层开始，逐层向前计算梯度：

1. 计算输出层的误差
2. 根据链式法则，计算上一层的误差
3. 重复直到输入层
4. 使用梯度更新参数

### 梯度下降

得到梯度后，使用梯度下降算法更新参数：

```
w = w - α * ∂J/∂w
b = b - α * ∂J/∂b
```

其中：
- `α`：学习率（Learning Rate），控制每次更新的步长

## 损失函数

### 什么是损失函数？

损失函数（Loss Function）用于衡量模型预测结果与真实值之间的差距。

### 常用损失函数

#### 1. 均方误差（Mean Squared Error，MSE）

```
MSE = (1/n) * Σ(y_pred - y_true)²
```

**应用场景**：回归任务

#### 2. 交叉熵损失（Cross-Entropy Loss）

**二分类**：
```
CrossEntropy = -y_true * log(y_pred) - (1-y_true) * log(1-y_pred)
```

**多分类**：
```
CrossEntropy = -Σy_true * log(y_pred)
```

**应用场景**：分类任务

## 优化算法

### 梯度下降变体

#### 1. 批量梯度下降（Batch Gradient Descent）

使用全部训练数据计算梯度：

**优点**：梯度准确，收敛稳定
**缺点**：计算量大，内存消耗大

#### 2. 随机梯度下降（Stochastic Gradient Descent，SGD）

使用单个样本计算梯度：

**优点**：计算快，内存消耗小
**缺点**：梯度噪声大，收敛不稳定

#### 3. 小批量梯度下降（Mini-batch Gradient Descent）

使用一小批样本计算梯度：

**优点**：兼顾计算效率和梯度准确性
**缺点**：需要选择合适的批次大小（Batch Size）

### 动量（Momentum）

动量算法在梯度下降中加入了"惯性"，加速收敛：

```
v = βv + α * ∂J/∂θ
θ = θ - v
```

其中，`β`是动量系数（通常为0.9）。

### AdaGrad

AdaGrad自适应调整学习率，对稀疏特征给予更大的学习率：

```
G = G + (∂J/∂θ)²
θ = θ - α / √G * ∂J/∂θ
```

### RMSprop

RMSprop改进了AdaGrad，使用指数移动平均：

```
E[g²] = βE[g²] + (1-β)(∂J/∂θ)²
θ = θ - α / √E[g²] * ∂J/∂θ
```

### Adam

Adam结合了动量和RMSprop的优点，是目前最常用的优化算法：

```
m = β₁m + (1-β₁)∂J/∂θ
v = β₂v + (1-β₂)(∂J/∂θ)²
m̂ = m / (1-β₁ᵗ)
v̂ = v / (1-β₂ᵗ)
θ = θ - α * m̂ / (√v̂ + ε)
```

其中：
- `β₁`：动量系数（通常为0.9）
- `β₂`：RMSprop系数（通常为0.999）
- `ε`：防止除以0的小常数（通常为1e-8）

## 批归一化

### 什么是批归一化？

批归一化（Batch Normalization，BN）是一种加速训练和提高稳定性的技术。

### 工作原理

对每一层的输入进行归一化处理：

```
μ = (1/m) * Σxᵢ
σ² = (1/m) * Σ(xᵢ - μ)²
x̂ᵢ = (xᵢ - μ) / √(σ² + ε)
yᵢ = γx̂ᵢ + β
```

其中：
- `μ`：批次均值
- `σ²`：批次方差
- `γ`和`β`：可学习的缩放和偏移参数

### 优点

1. 加速收敛（不需要仔细调整学习率）
2. 缓解梯度消失
3. 减少过拟合（相当于正则化）

## 神经网络架构

### 全连接神经网络（Feedforward Neural Network，FNN）

全连接神经网络中，每一层的神经元与下一层的所有神经元相连。

**应用场景**：简单的分类和回归任务

### 卷积神经网络（Convolutional Neural Network，CNN）

详见：[计算机视觉.md](计算机视觉.md)

### 循环神经网络（Recurrent Neural Network，RNN）

RNN专门用于处理序列数据，它在时间维度上共享参数：

```
hₜ = σ(Wₓh xₜ + Wₕh hₜ₋₁ + bₕ)
yₜ = σ(Wₕy hₜ + bᵧ)
```

**问题**：
- 梯度消失（难以学习长期依赖）
- 梯度爆炸

### 长短期记忆网络（Long Short-Term Memory，LSTM）

LSTM是一种改进的RNN，通过门机制解决了梯度消失问题：

**门结构**：
- **遗忘门（Forget Gate）**：决定保留哪些历史信息
- **输入门（Input Gate）**：决定输入哪些新信息
- **输出门（Output Gate）**：决定输出哪些信息

**记忆单元（Cell State）**：类似于传送带，直接传递信息

## 正则化技术

### 权重衰减（Weight Decay）

在损失函数中加入权重的L2范数：

```
Loss = CrossEntropy + λ * Σw²
```

### Dropout

Dropout在训练过程中随机丢弃一部分神经元（设为0）：

**工作原理**：
- 训练时：每个神经元有p概率被保留
- 测试时：所有神经元都保留，但输出乘以p

**优点**：防止过拟合，提高模型泛化能力

### 数据增强（Data Augmentation）

通过对训练数据进行变换来增加数据多样性：

- 图像：旋转、翻转、裁剪、缩放
- 文本：同义词替换、随机删除、随机插入

## 超参数调优

### 什么是超参数？

超参数是在训练前设置的参数，不是通过训练得到的：

- 学习率（Learning Rate）
- 批次大小（Batch Size）
- 隐藏层数量
- 每层神经元数量
- Dropout率

### 调优方法

#### 网格搜索（Grid Search）

穷举所有可能的超参数组合，选择最优的组合。

**缺点**：计算量大

#### 随机搜索（Random Search）

随机采样超参数组合，在一定次数内找到较好的组合。

**优点**：比网格搜索更高效

#### 贝叶斯优化（Bayesian Optimization）

基于贝叶斯定理，根据已有结果预测下一个最优超参数组合。

**优点**：高效，适合高维空间

## 训练技巧

### 学习率调度（Learning Rate Scheduling）

根据训练进度调整学习率：

- **阶梯式衰减**：每训练一定轮次，学习率乘以一个衰减因子
- **余弦退火**：学习率按余弦函数周期性变化
- **预热（Warmup）**：初始学习率很小，逐渐增大

### 梯度裁剪（Gradient Clipping）

防止梯度爆炸，将梯度的范数限制在一定范围内：

```
if ||grad|| > max_norm:
    grad = grad * max_norm / ||grad||
```

### 早停（Early Stopping）

当验证集性能不再提升时，提前停止训练，防止过拟合：

1. 记录验证集性能（如损失或准确率）
2. 如果连续N个epoch没有提升，停止训练
3. 使用性能最好的模型
