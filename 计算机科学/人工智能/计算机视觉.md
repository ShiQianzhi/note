# 计算机视觉

## 什么是计算机视觉？

**计算机视觉（Computer Vision，CV）** 是人工智能的一个分支，旨在让计算机能够"看"和理解图像及视频。

### CV的目标

让计算机能够：
- **识别**：识别图像中的物体
- **检测**：定位图像中的物体
- **分割**：将图像中的不同区域分开
- **生成**：生成新的图像

### 应用场景

- 人脸识别（解锁手机、安防监控）
- 自动驾驶（道路识别、障碍物检测）
- 医疗影像诊断（X光片、CT扫描分析）
- 图像搜索（以图搜图）
- 图像生成（DALL-E、Midjourney）

## 图像处理基础

### 图像表示

#### 数字图像

数字图像由像素（Pixel）组成，每个像素包含颜色信息：

- **灰度图像**：每个像素用0-255表示亮度（0=黑，255=白）
- **彩色图像**：每个像素包含红、绿、蓝三个通道（RGB）

#### 图像格式

- **JPEG**：有损压缩，适合照片
- **PNG**：无损压缩，支持透明
- **BMP**：无压缩，文件大
- **GIF**：支持动画，颜色有限

### 图像预处理

#### 1. 缩放（Resizing）

调整图像大小：
- 放大（Upsampling）：增加像素
- 缩小（Downsampling）：减少像素

#### 2. 裁剪（Cropping）

截取图像的一部分区域。

#### 3. 旋转（Rotation）

旋转图像一定角度。

#### 4. 翻转（Flipping）

水平翻转或垂直翻转图像。

#### 5. 归一化（Normalization）

将像素值从0-255转换为0-1或-1-1：

```python
# 方法1：除以255
normalized = image / 255.0

# 方法2：标准化
normalized = (image - mean) / std
```

#### 6. 数据增强（Data Augmentation）

通过变换增加训练数据多样性：
- 随机裁剪
- 随机翻转
- 随机旋转
- 随机亮度/对比度调整
- 随机噪声添加

## 卷积神经网络

### 为什么需要CNN？

传统神经网络使用全连接层处理图像，存在以下问题：

1. **参数过多**：一张224x224的RGB图像有150528个像素，全连接层需要大量参数
2. **空间信息丢失**：全连接层忽略了像素之间的空间关系

CNN通过局部连接和参数共享解决了这些问题。

### 卷积操作

#### 什么是卷积？

卷积是一种数学运算，用于提取图像的特征：

```
输出[i,j] = ΣΣ 输入[i+k,j+l] * 卷积核[k,l]
```

#### 卷积核（Kernel/Filter）

卷积核是一个小矩阵，用于检测特定特征：

- **边缘检测核**：检测图像中的边缘
- **模糊核**：平滑图像
- **锐化核**：增强图像细节

#### 感受野（Receptive Field）

每个输出像素对应的输入区域称为感受野。

#### 步长（Stride）

卷积核在输入上滑动的步长。

#### 填充（Padding）

在输入图像周围填充0，保持输出大小与输入相同：

- **Valid**：不填充，输出变小
- **Same**：填充，输出大小不变

### 池化操作

#### 什么是池化？

池化是一种下采样操作，用于减少特征图的大小：

- **最大池化（Max Pooling）**：取区域内的最大值
- **平均池化（Average Pooling）**：取区域内的平均值

#### 作用

1. 减少计算量
2. 增加平移不变性
3. 防止过拟合

### CNN架构

#### 典型结构

```
输入 → 卷积层 → 激活函数 → 池化层 → ... → 全连接层 → 输出
```

#### 各层作用

1. **卷积层**：提取特征（边缘、纹理、形状等）
2. **激活函数**：引入非线性
3. **池化层**：减少特征图大小
4. **全连接层**：分类或回归

#### 经典CNN架构

##### LeNet-5（1998年）

最早的CNN之一，用于手写数字识别：
- 2个卷积层
- 2个池化层
- 3个全连接层

##### AlexNet（2012年）

在ImageNet比赛中取得突破性成绩：
- 8层网络
- 使用ReLU激活函数
- 使用Dropout防止过拟合
- 使用GPU加速

##### VGGNet（2014年）

采用小卷积核（3x3）和深网络：
- 16-19层网络
- 连续使用3x3卷积
- 结构简洁统一

##### GoogLeNet/Inception（2014年）

引入Inception模块，多尺度特征提取：
- 22层网络
- Inception模块并行使用不同大小的卷积核

##### ResNet（2015年）

引入残差连接，解决深度网络训练困难：
- 残差连接：x + F(x)
- 可以训练超过100层的网络

##### EfficientNet（2019年）

通过缩放网络深度、宽度和分辨率来优化性能。

## 图像分类

### 什么是图像分类？

图像分类是将图像分类到预定义类别中的任务：

```
输入：一张猫的图片
输出："猫"（概率：95%）
```

### 训练流程

1. 准备数据集（如ImageNet）
2. 数据预处理和增强
3. 选择或构建CNN模型
4. 训练模型（使用交叉熵损失）
5. 评估模型（使用准确率）

### 评估指标

- **Top-1准确率**：预测的最可能类别正确的比例
- **Top-5准确率**：预测的前5个类别中包含正确类别

## 目标检测

### 什么是目标检测？

目标检测不仅要识别图像中的物体，还要定位它们的位置：

```
输入：一张包含猫和狗的图片
输出：[
    {"类别": "猫", "位置": [x1, y1, x2, y2]},
    {"类别": "狗", "位置": [x3, y3, x4, y4]}
]
```

### 检测框表示

通常使用边界框（Bounding Box）表示物体位置：
- **(x1, y1)**：左上角坐标
- **(x2, y2)**：右下角坐标

### 目标检测算法

#### 两阶段检测（Two-stage Detection）

先生成候选区域，再分类：

##### R-CNN（2014年）

1. 使用选择性搜索生成2000个候选区域
2. 对每个候选区域提取特征
3. 使用SVM分类
4. 使用回归器微调边界框

##### Fast R-CNN（2015年）

改进R-CNN，共享特征计算：
1. 对整张图像提取特征
2. 使用感兴趣区域池化（RoI Pooling）

##### Faster R-CNN（2015年）

使用区域提议网络（RPN）代替选择性搜索：
1. RPN生成候选区域
2. Fast R-CNN进行分类和回归

#### 单阶段检测（One-stage Detection）

直接预测物体类别和位置：

##### YOLO（You Only Look Once）

将检测视为回归问题：
1. 将图像划分为SxS网格
2. 每个网格预测B个边界框和类别概率
3. 非极大值抑制（NMS）去除重叠框

**特点**：速度快，适合实时检测

##### SSD（Single Shot MultiBox Detector）

在多个特征图上进行检测：
- 底层特征图检测小物体
- 顶层特征图检测大物体

##### RetinaNet（2017年）

引入焦点损失（Focal Loss）解决类别不平衡问题。

## 图像分割

### 什么是图像分割？

图像分割是将图像划分为不同区域的任务：

- **语义分割（Semantic Segmentation）**：只区分类别，不区分个体
- **实例分割（Instance Segmentation）**：区分不同个体

### 语义分割算法

#### U-Net（2015年）

医学图像分割的经典模型：
- 编码器：下采样提取特征
- 解码器：上采样恢复空间信息
- 跳跃连接：融合高低层特征

#### DeepLab系列

使用空洞卷积（Atrous Convolution）扩大感受野：
- DeepLabv1：使用空洞卷积
- DeepLabv2：引入Atrous Spatial Pyramid Pooling（ASPP）
- DeepLabv3：改进ASPP和编码器
- DeepLabv3+：添加解码器模块

## 图像生成

### 什么是图像生成？

图像生成是根据输入生成新图像的任务。

### 生成对抗网络（GAN）

#### GAN结构

GAN包含两个网络：

1. **生成器（Generator）**：从随机噪声生成图像
2. **判别器（Discriminator）**：判断图像是真实的还是生成的

#### 训练过程

1. 生成器生成假图像
2. 判别器区分真假图像
3. 生成器试图欺骗判别器
4. 判别器试图正确区分

**目标函数**：
```
min_G max_D V(D, G) = E[log D(x)] + E[log(1 - D(G(z)))]
```

#### GAN变体

- **DCGAN**：使用CNN的GAN
- **StyleGAN**：控制生成图像的风格
- **CycleGAN**：无配对图像翻译

### 变分自编码器（VAE）

VAE是一种生成模型，使用编码器和解码器：

1. 编码器将图像编码为潜在空间的分布
2. 从分布中采样得到潜在向量
3. 解码器将潜在向量解码为图像

### 扩散模型（Diffusion Model）

扩散模型是目前最先进的图像生成模型：

#### 前向过程

逐步向图像添加高斯噪声，直到完全变成噪声。

#### 反向过程

逐步从噪声中恢复出清晰的图像。

**代表模型**：
- DALL-E
- Stable Diffusion
- Midjourney

## 人脸识别

### 什么是人脸识别？

人脸识别是识别图像中人脸身份的任务。

### 流程

1. **人脸检测**：定位图像中的人脸
2. **人脸对齐**：将人脸调整到标准姿态
3. **特征提取**：提取人脸特征向量
4. **特征比对**：与数据库中的人脸进行比对

### 度量学习（Metric Learning）

人脸识别使用度量学习，让同一人的特征距离近，不同人的特征距离远：

- **Triplet Loss**：锚点-正样本-负样本
- **Center Loss**：类内紧凑，类间分离

### 典型模型

- **FaceNet**：使用Triplet Loss训练
- **ArcFace**：使用ArcFace Loss，提升识别精度

## 评估指标

### 图像分类

- **Top-1准确率**
- **Top-5准确率**

### 目标检测

#### IoU（Intersection over Union）

衡量预测框与真实框的重叠程度：

```
IoU = (预测框 ∩ 真实框) / (预测框 ∪ 真实框)
```

**阈值**：通常设置为0.5或0.75

#### mAP（mean Average Precision）

平均精确率的平均值，综合衡量检测性能。

### 图像分割

#### IoU

与目标检测相同，衡量预测分割与真实分割的重叠程度。

#### Dice系数

```
Dice = 2 * (预测 ∩ 真实) / (|预测| + |真实|)
```
