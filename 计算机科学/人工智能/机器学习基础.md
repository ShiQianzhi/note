# 机器学习基础

## 什么是机器学习？

**机器学习（Machine Learning，ML）** 是人工智能的一个分支，它赋予计算机从数据中学习的能力，而无需进行明确的编程。

### 核心思想

机器学习的核心思想是：**通过数据训练模型，让模型自动发现规律并做出预测或决策。**

### 类比理解

想象你教一个小孩认识苹果：

1. 你给小孩看很多苹果的图片（数据）
2. 小孩观察苹果的特征：红色、圆形、甜的（特征提取）
3. 小孩总结出苹果的规律（模型学习）
4. 下次看到新的苹果，小孩能认出来（预测）

这就是机器学习的基本过程。

## 机器学习的分类

### 按学习方式分类

#### 1. 监督学习（Supervised Learning）

**监督学习**是最常见的学习方式。就像老师教学生一样，我们给模型提供**标注好的数据**（输入+正确输出），让模型学习输入和输出之间的映射关系。

**典型应用：**
- **分类（Classification）**：判断输入属于哪个类别
  - 垃圾邮件识别（垃圾邮件/正常邮件）
  - 图像识别（猫/狗/鸟）
- **回归（Regression）**：预测一个连续的数值
  - 房价预测
  - 股票价格预测

**常用算法：**
- 线性回归（Linear Regression）
- 逻辑回归（Logistic Regression）
- 决策树（Decision Tree）
- 随机森林（Random Forest）
- 支持向量机（Support Vector Machine，SVM）
- K近邻（K-Nearest Neighbors，KNN）

#### 2. 无监督学习（Unsupervised Learning）

**无监督学习**不给模型提供标注数据，只给输入数据，让模型自己发现数据中的规律和结构。

**典型应用：**
- **聚类（Clustering）**：将相似的数据分组
  - 客户分群（根据购买行为）
  - 图片分类（没有标签）
- **降维（Dimensionality Reduction）**：减少数据维度，便于可视化和计算
  - 主成分分析（PCA）
  - t-SNE

**常用算法：**
- K均值聚类（K-Means）
- 层次聚类（Hierarchical Clustering）
- 主成分分析（Principal Component Analysis，PCA）
- 关联规则挖掘（Association Rule Mining）

#### 3. 半监督学习（Semi-supervised Learning）

**半监督学习**结合了监督学习和无监督学习，使用少量标注数据和大量未标注数据进行训练。

**典型应用：**
- 图像识别（少量标注图片 + 大量未标注图片）
- 文本分类

#### 4. 强化学习（Reinforcement Learning）

**强化学习**让智能体（Agent）在与环境的交互中学习。智能体通过执行动作获得奖励或惩罚，从而学习最优策略。

详见：[强化学习.md](强化学习.md)

## 机器学习工作流程

### 完整流程

```
数据收集 → 数据预处理 → 特征工程 → 模型选择 → 模型训练 → 模型评估 → 模型部署
```

#### 1. 数据收集

收集与任务相关的数据，数据质量直接影响模型效果。

#### 2. 数据预处理

原始数据通常存在问题，需要进行预处理：

- **数据清洗**：处理缺失值、异常值、重复数据
- **数据转换**：标准化、归一化、编码（如独热编码）
- **数据划分**：训练集（Training Set）、验证集（Validation Set）、测试集（Test Set）

**训练集/验证集/测试集划分：**
- **训练集（70%-80%）**：用于模型训练
- **验证集（10%-15%）**：用于调参和选择模型
- **测试集（10%-15%）**：用于最终评估模型性能

#### 3. 特征工程

特征工程是机器学习中最重要的环节之一，它决定了模型的上限。

**特征选择**：选择对目标变量最有影响力的特征
**特征提取**：从原始数据中提取有意义的特征
**特征转换**：对特征进行变换（如对数变换、多项式变换）

#### 4. 模型选择

根据任务类型和数据特点选择合适的模型：

- 回归任务：线性回归、随机森林回归、XGBoost
- 分类任务：逻辑回归、SVM、随机森林、神经网络
- 聚类任务：K-Means、DBSCAN

#### 5. 模型训练

使用训练集数据训练模型，调整模型参数以最小化损失函数。

#### 6. 模型评估

使用测试集评估模型性能，常用指标见下文。

#### 7. 模型部署

将训练好的模型部署到生产环境，提供预测服务。

## 评估指标

### 分类任务指标

#### 混淆矩阵（Confusion Matrix）

```
              预测值
            正例    负例
真实值 正例  TP      FN
      负例  FP      TN
```

- **TP（True Positive）**：真实为正，预测为正（正确）
- **FN（False Negative）**：真实为正，预测为负（漏检）
- **FP（False Positive）**：真实为负，预测为正（误报）
- **TN（True Negative）**：真实为负，预测为负（正确）

#### 准确率（Accuracy）

```
准确率 = (TP + TN) / (TP + TN + FP + FN)
```

预测正确的样本占总样本的比例。

**局限性**：在不平衡数据上表现不佳。例如，99%的样本是负例，模型全预测负例也能得到99%的准确率。

#### 精确率（Precision）

```
精确率 = TP / (TP + FP)
```

预测为正的样本中，真正为正的比例。

**应用场景**：垃圾邮件识别（宁愿放过垃圾邮件，也不要误删正常邮件）。

#### 召回率（Recall）

```
召回率 = TP / (TP + FN)
```

真正为正的样本中，被预测为正的比例。

**应用场景**：癌症诊断（宁愿误判健康人，也不要漏掉癌症患者）。

#### F1分数（F1-Score）

```
F1 = 2 * (精确率 * 召回率) / (精确率 + 召回率)
```

精确率和召回率的调和平均数，综合衡量模型性能。

#### ROC曲线和AUC

**ROC曲线（Receiver Operating Characteristic Curve）**：以假阳性率（FPR）为横轴，真阳性率（TPR）为纵轴绘制的曲线。

**AUC（Area Under Curve）**：ROC曲线下的面积，取值范围0.5-1，值越大表示模型性能越好。

### 回归任务指标

#### 均方误差（Mean Squared Error，MSE）

```
MSE = (1/n) * Σ(y_pred - y_true)²
```

预测值与真实值差的平方的平均值。

#### 平均绝对误差（Mean Absolute Error，MAE）

```
MAE = (1/n) * Σ|y_pred - y_true|
```

预测值与真实值差的绝对值的平均值。

#### R²分数（R-Squared）

```
R² = 1 - (Σ(y_true - y_pred)²) / (Σ(y_true - y_mean)²)
```

模型解释的方差占总方差的比例，取值范围0-1，越接近1表示模型拟合效果越好。

## 常见机器学习算法

### 线性回归（Linear Regression）

线性回归是最简单的回归算法，假设输出与输入之间存在线性关系：

```
y = w₁x₁ + w₂x₂ + ... + wₙxₙ + b
```

其中，`w`是权重（Weight），`b`是偏置（Bias）。

**应用场景**：房价预测、销售额预测。

### 逻辑回归（Logistic Regression）

逻辑回归虽然名字叫"回归"，但实际上是用于分类任务的算法。它使用sigmoid函数将线性输出映射到0-1之间：

```
σ(z) = 1 / (1 + e^(-z))
```

其中，`z = w·x + b`。

**应用场景**：垃圾邮件识别、疾病诊断。

### 决策树（Decision Tree）

决策树是一种基于树结构的分类和回归算法，它通过一系列决策规则对数据进行划分。

**优点**：
- 易于理解和解释
- 可以处理非线性数据
- 不需要特征缩放

**缺点**：
- 容易过拟合
- 对噪声敏感

### 随机森林（Random Forest）

随机森林是一种集成学习算法，它构建多个决策树并综合它们的预测结果。

**集成学习思想**：三个臭皮匠，顶个诸葛亮。

**优点**：
- 抗过拟合能力强
- 泛化能力好
- 可以处理高维数据

### 支持向量机（Support Vector Machine，SVM）

SVM是一种强大的分类算法，它寻找最优超平面来分隔不同类别的数据。

**核技巧（Kernel Trick）**：将数据映射到高维空间，使其线性可分。

**常用核函数**：
- 线性核（Linear Kernel）
- 多项式核（Polynomial Kernel）
- 高斯核（Gaussian Kernel，也叫RBF核）

**应用场景**：文本分类、图像识别。

### K近邻（K-Nearest Neighbors，KNN）

KNN是一种简单但有效的算法，它根据最近的K个邻居来预测新样本的类别。

**工作原理**：
1. 计算新样本与所有训练样本的距离
2. 选择距离最近的K个样本
3. 根据这K个样本的类别进行投票，决定新样本的类别

**优点**：
- 简单易懂
- 无需训练过程

**缺点**：
- 计算量大
- 对K值敏感

## 过拟合与欠拟合

### 过拟合（Overfitting）

**过拟合**是指模型在训练数据上表现很好，但在测试数据上表现很差的现象。

**原因**：
- 模型过于复杂（如深度过深的决策树）
- 训练数据过少或噪声过多

**表现**：
- 训练集准确率很高，测试集准确率很低

**解决方法**：
- **增加训练数据**
- **简化模型**（减少特征、降低模型复杂度）
- **正则化**（L1正则化、L2正则化）
- **交叉验证**

### 欠拟合（Underfitting）

**欠拟合**是指模型过于简单，无法捕捉数据中的规律。

**表现**：
- 训练集和测试集的准确率都很低

**解决方法**：
- **增加模型复杂度**（增加特征、使用更复杂的模型）
- **特征工程**（提取更多有意义的特征）

## 正则化

### L1正则化（Lasso Regression）

```
损失函数 = MSE + λ * Σ|wᵢ|
```

L1正则化会使一些权重变为0，起到特征选择的作用。

### L2正则化（Ridge Regression）

```
损失函数 = MSE + λ * Σwᵢ²
```

L2正则化会使权重变小，但不会变为0，防止权重过大。

### λ参数

- `λ = 0`：无正则化
- `λ`越大：正则化效果越强，模型越简单

## 交叉验证

### 什么是交叉验证？

交叉验证是一种评估模型性能的方法，它将数据集划分为多个子集，轮流用其中一个子集作为验证集，其余作为训练集。

### K折交叉验证（K-fold Cross Validation）

1. 将数据集划分为K个大小相等的子集
2. 依次用第i个子集作为验证集，其余K-1个子集作为训练集
3. 重复K次，得到K个评估结果
4. 取平均值作为最终评估结果

**常用K值**：5或10
